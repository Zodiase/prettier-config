# `@xch/prettier-config`

> My personal [Prettier](https://prettier.io) config

## Usage

**Install**:

```bash
$ npm install --dev @xch/prettier-config
```

**Edit `package.json`**:

```jsonc
{
    // ...
    "prettier": "@xch/prettier-config"
}
```
